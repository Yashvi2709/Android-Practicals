# Practical 13: Contacts and SMS

## Objective
To read contacts and send SMS using an Android application.

## Description
An Android application was developed to read contact names from the phone using Content Provider and display them in a ListView. On long press of a contact, a context menu appears with an option "Send SMS". On selecting this option, the messaging application opens with the selected contact number.

## Components Used
- Content Provider  
- ListView  
- Context Menu  
- Intent
