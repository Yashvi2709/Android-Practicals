# Practical 14: SQLite Orders Application

## Objective
To store and retrieve data using SQLite database.

## Description
An Android application was developed for a salesman to take and view orders. The "Take Order" activity allows the user to enter phone number, item name, and quantity. On clicking Save, the data is stored in SQLite database. The "View Order" activity displays all saved orders in a ListView.

## Components Used
- SQLite Database  
- EditText  
- Button  
- ListView  
- Intent
