# Practical 11: Login Application with SharedPreferences

## Objective
To implement login functionality and store data using SharedPreferences.

## Description
An Android application was developed with Email, Password, and "Remember Me" option. On clicking the Login button, a new activity is opened showing a welcome message. If "Remember Me" is checked, the entered data is stored using SharedPreferences.

## Components Used
- EditText  
- Button  
- CheckBox  
- SharedPreferences  
- Intent
