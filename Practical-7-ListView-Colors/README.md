# Practical 7: ListView Color Changer

## Objective
To display a list of colors and change background color on selection.

## Description
An Android application was developed that displays a list of color names using ListView. When a user clicks on any color name, the background color of the activity changes accordingly.

## Components Used

- OnItemClickListener
