# Practical 3: User Input Form

## Objective
To collect user information and display it in a new activity.

## Description
An Android application was developed to collect user details such as Username, Password, Email, Phone Number, Country, State, Gender, Interests, Birth Date, and Time. When the user clicks the Submit button, the entered data is sent to another activity using Intent and displayed.

## Components Used
- EditText  
- RadioButton  
- CheckBox  
- Button  
- Intent
